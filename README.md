
### Quick start
**Make sure you have Node version >= 5.0 and NPM >= 3**

```bash
# clone the repo

# install the repo with npm
npm install

# build client directory for production
npm run build:prod

# run the server
gulp serve
```

^ All above commands has an alias command which is: 

```
npm start
```

***

for development use:

```bash
# clone the repo

# install the repo with npm
npm install

# build client directory for production
npm run watch:dev:hmr

# run the server
gulp serve
